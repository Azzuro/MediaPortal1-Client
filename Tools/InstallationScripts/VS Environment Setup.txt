

1. Update to Windows SDK v6.1:

        http://www.microsoft.com/downloads/details.aspx?FamilyId=F26B1AA4-741A-433A-9BE5-FA919850BDBF&displaylang=en




2. Create the following registry key (check the path if you did a custom install):

	Windows Registry Editor Version 5.00

	[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework]
	"SDKInstallRootv2.0"="C:\\Program Files\\Microsoft SDKs\\Windows\\v6.1"




3. If you have a x64 system, check "\svnroot\mediaportal\trunk\Tools\InstallationScripts\DeployToolUnPacker.nsi" for the above line:

	!system '"$%WINDIR%\Microsoft.NET\Framework\v3.5\MSBUILD.exe" /p:ALToolPath="%WINDOWS_SDK%\Bin" /target:Rebuild /property:Configuration=Release;Platform=x86 "${svn_DeployTool}\MediaPortal.DeployTool.sln"' = 0

  and change the ALToolPath parameter to /p:ALToolPath="%WINDOWS_SDK%\Bin\x64"




NOTE: Used http://msdn.microsoft.com/en-us/windowsserver/bb986638.aspx, chapter 5.1.2 as reference
