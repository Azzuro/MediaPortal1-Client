

									MediaPortal Setup 1.0RC2


To create the selfextracting package you need WinRAR >= 3.71.
Take a look at WinRAR_*.jpg for build options.

Files to include:

   MediaPortal.DeployTool.exe											<- DeployTool exe
   Interop.NetFwTypeLib.dll												<- WindowsFirewall dll from XP SP2
   *\MediaPortal.DeployTool.resources.dll					<- DeployTool's localizations
   deploy\package-mediaportal.exe									<- Full MP  install package
   deploy\package-tvengine.exe										<- Full TVE install package
   deploy\SVNxxxxx																<- A empty file named with the SVN build included
