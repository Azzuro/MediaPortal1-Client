MediaPortal 1 Setup Guide for Beginners

Welcome to the MediaPortal Setup Guide. This guide has been written to ensure your first experience with MediaPortal will be quick and painless. It includes not all but the most common sceneraios for a MediaPortal installation.

  • Note: Due to external links this page is best viewed with an active internet connection.

Step 1 :- Before You Begin

  • Before you begin the configuration of MediaPortal, make sure you have the following steps completed:

  • Ensure your system complies the requirements. As MediaPortal is developed based on the latest updates and hotfixes for Microsoft sofware, it is essential that your Windows installation is up to date with the latest service packs and hotfixes. To achive this simply run Windows update.

  • For hotfixes which are not part of the Windows Update System, refer to the requirements.

  • Install the latest drivers for your hardware (e.g. video card, TV card,...).

  • For Windows XP users that wish to use TV cards, you have to install the Microsoft DVB Hotfix to ensure smooth operation. The hotfix itself and more information about it can be found in the MediaPortal forum

  • Now you can complete the installation of MediaPortal. For indepth installation instructions, please review MediaPortal Deploy Tool Manual

  • IMPORTANT NOTE:

  • For a MediaPortal installation there are 2 types of TV engines available. During the installation you were asked a question regarding the type of TV content you would like to watch.If you selected to watch HDTV during the installation process, then complete Step 2 :- TV Server Setup first, otherwise you can skip it and start with Step 3 :- MediaPortal Setup. For more information on the two different TV engines please visit TV Server Overview.

Step 2 : TV Server Setup

  • After the installation of MediaPortal, start the TV-Server Configuration (a shortcut is placed onto your desktop).

  • When starting the configuration for the first time you are asked for the database password which you chose

    during the installation (default is MediaPortal). Enter it and click Yes in the next popup to start the TV service.

  • On the left hand menu side, browse to TV Servers/Machine Name and ensure all cards that have been installed in your computer are listed.

  • Select a displayed TV card and select your country/area on the right hand side(this sets a predefined list of known channels in your area and speeds up the scanning process). If you are using a CAM you additionally have to select its type (this is ignored for DVB-T cards). Once this have been configured, click the Scan for Channels button. The scanning process will now take about 5 to 30 minutes depending on the number of channels in your country. Complete this step for all of your TV cards.

  • If you have a hybrid TV card (both digital and analogue TV tuners on one card), you need to click the hybrid tab to setup a grouping to ensure resources are always available when switching between digital and analogue channels. More information about hybrid cards can be found in Configuring Hybrid cards.

  • If you want to obtain a program guide from the digital TV signal (EPG), then select General Settings/ EPG tab and enable it.

  • You can now move on with the configuration of the recording settings. Select Recording Settings/Settings tab and specify how many minutes before the program actually starts you want the recording to begin. (great for channels that don't stick to advertised times in their guides). Also specify how long you want the recording to continue after the program has finished.

  • Next is to set up the location of the recordings and the timeshifting files. Select Recording settings/Recording Folders tab. Now select each card installed in your system and change the recording path to your liking. Here you can also select the data format of the recorded files (ts stream or MPEG, ts is preferred)

  • A more detailed installation guide can be found in the MediaPortal wiki under TV Server Manual.

Step 3 :- MediaPortal Setup

  • After installing MediaPortal, start MediaPortal Configuration (a shortcut is placed onto your desktop).

  • Browse to Movies/Movie Folders and specify the location of your movie collection.

  • Browse to Movies/Movie Player and specify your preferred codecs. If you are unsure of what codecs are, select MPV for video and MPA for audio.

  • Browse to DVD/DVD Codecs and specify your favored codecs.

  • Browse to Music/Folders and specify the location of your music collection.

  • Browse to Pictures/Folders and specify the location of your picture collection.

  • Browse to Television and specify your preferred codecs.

  • If you have selected to install the inbuilt TV Engine then continue with the Step 4, otherwise you can skip this step.

Step 4 :- Inbuilt TV Engine

  • Goto Television/Capture Cards and add your capture cards if they have not been automatically recognised.

  • Goto Televsion/TV Channels and look up your TV channels. This will import the channels for your area. If not all channels were found, you can manually add the missing ones. Visit the MediaPortal Wiki for more information.

  • Goto to Television/Program Guide and

    • for DVB type cards, select "Grab EPG for DVB".
    • for Analogue and others, setup a XMLTV source guide from the internet.

  • Goto Television/Recording and set your pre-record and post record timings for your scheduled programs.

  • A more detailed installation guide can be found on the MediaPortal wiki under MediaPortal Manual

Summary

  • Having finished the basic configuration of MediaPortal you are now ready to enjoy your media collection. If you want to further customise your MediaPortal installation, you can have a look at theMediaPortal Wiki for more detailed documentation.
